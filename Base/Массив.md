## Массив - это
Набор проиндексированных переменных определенного размера.

данные хранатятся последовательно.

например:
int array1 :=   {1, 2, 3, 4, 5}
            ^  ^   ^   ^  ^
индексация   0  1  2   3  4

Массив обычно строго определен по обьёму.
