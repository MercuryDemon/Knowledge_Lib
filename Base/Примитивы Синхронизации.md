# Примитивы синхронизации

## mutex
- Самый базовый примитив чтобы избежать одновременного чтения и записи в одну область памяти

## RWmutex (Read Write Mytex)
- Позволяет множеству горутин одновременно читать, но только одной писать.

## WaitGroup
- Ждем завершения всех воркеров

## Once
- Гарантирует, что функция выполнится только один раз, даже при вызове из множества горутин.

## Cond
- условная переменная, которая позволяет горутинам ждать наступления события.

## syncPOOL
- пул временных объектов. Позволяет переиспользовать объекты, уменьшая нагрузку на GC

## syncMap
- Потокобезопасная мапа

## Каналы
- Обмен данными между горутинами

## Atomic
- Для простых счетчиков и флагов можно использовать атомарные операции без блокировок.
